Word Rush is a fast-typing game created for the Final Project assignment. Players type randomized words within a 99-second timer, and the game tracks hits and accuracy.
---
## üéÆ How to Play
1. Click **Start**.
2. Type the displayed word and press Enter.
3. Continue typing as many words as possible before the timer expires.
4. View your score and start a new game.

## Features
The game uses 90 words, and they appear in a random order each time.

Players get 99 seconds to type as many words as they can.

A new word shows up right away after you type the correct one.

A Start button is used to begin or restart the game without refreshing the page.

Background music plays during the game and stops when the game is over.

At the end, the game shows your score, including how many words you got right and your accuracy percentage.
